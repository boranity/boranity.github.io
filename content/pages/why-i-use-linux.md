I can occasionally post articles in English. In this post i will briefly talk about why i use linux.

# FOSS (Free And Open Source)
I prefer free software, i like the philosophy, i prefer open source for most of my applications and systems.

# Pleasures
Using a terminal makes my work easier. Configuring systems and similar things are a pleasure for me.

# Getting rid of NTFS garbage
NTFS garbage is a disk killing file systems and im so glad i dont use it. I recently started using btrfs when btrfs improvements came for kernel 6.1

# Privacy
Privacy is important, Im not as obsessed with it as I used to be, but still, nobody wants to use a system that fucks with privacy, right lul?

# Developing
I'd say its really the best for writing code and most people would agree. I dont prefer C# garbage. :D

# Which distribution do I prefer
I like Arch Linux and most of my Linux experience has been with Arch/Based distributions.

# Sysadmin
Sysadmin is a job i enjoy and its even more fun with *NIX.
