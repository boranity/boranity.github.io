# Line Number Açma

74-75 satırda olan `# set linenumbers`'ın başındaki # etiketini kaldırın.
```
 ## Display line numbers to the left (and any anchors in the margin).
    set linenumbers
```
Böyle gözükecek. Eğer kısayol kullanmak istiyorsanız Alt + Altgr + 3 kombinasyonunu uygulayabilirsiniz.

# Syntax Highlighting Açma

241-242 satırda olan `# include "/usr/share/nano/*.nanorc"` başındaki etiketi kaldırın.
```
 ## To include most of the existing syntax definitions, you can do:
    include "/usr/share/nano/*.nanorc"
```
Böyle gözükecek sorun çözülmez ise benimle telegram üzerinden iletişim kurabilirsiniz.
